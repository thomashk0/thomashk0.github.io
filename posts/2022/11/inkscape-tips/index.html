Some Inkscape Tips

As a researcher, having some basic knowledge of a vector drawing tool is very helpful. Inkscape is an incredible open-source project, which among others allows creating and editing SVG files.

Because Inkscape is very generic and not tuned for creating specific diagrams, it can take some efforts. The first drawings (including mines at the time) usually look, well, "artistic" lets say. So, those days, many people will perfer more friendly tools like draw.io, which generate clean generate with little efforts.

The learing curve of Inkscape is steep, but I would argue that it us worth investing time in it. With a bit of practice you can basically produce any kind of diagram (more that enough for scientific papers). In this post I show some tip to improve the Inkscape experience.

Note

This goal of this post is not to convince you to switch to Inkscape . I do not like draw.io for several reasons. But I strongly belive that the useful tools are the one you masters.

Basic Configuration

When you want to draw properly aligned elements, I find the grid mandatory. You want to do two things:

  • Display the grid (Press "#" for switching it on/off). You can customize the grid, I usally don't find it useful.
  • Enable snapping (Press "%" for switching it on/off), so that items like boxes will automatically snap to the grid.

This way, you can ensure consistent spacing of items.

Conversion

For converting SVG to other formats, I usually export the drawing region. This requires you to properly resize the document to your drawing area.

Export to PDF:

$ inkscape --export-text-to-path -D -o output.pdf input.svg

The --export-text-to-path option is very important. As suggested, it transforms text to paths. Otherwise, text is rendered by the PDF rendering engine and font may be missing. By passing this flag, you will have a consistent rendering of your drawing.

Export to PNG:

$ inkscape -D -d 300 -o output.png input.svg

Here, the DPI parameter is set 300.

With Release 1.0, the Inkscape command line changed. Here are the versions of those commands for older Inkscape.

$ inkscape --export-text-to-path -D -A output.pdf input.svg
$ inkscape -D -d 300 -e output.png input.svg

Makefile Receipe

INKSCAPE ?= inkscape
INKSCAPE_FLAGS ?=
INKSCAPE_FLAGS += --export-text-to-path
INKSCAPE_VERSION_1 := $(shell inkscape --version | grep -o -E 'Inkscape 1\.[0-9]+')

figs/%.pdf: figs/svg/%.svg
ifeq ($(INKSCAPE_VERSION_1),) # Inkscape < 1.0
    $(INKSCAPE) $(INKSCAPE_FLAGS) -D -A $@ $<
else
    $(INKSCAPE) $(INKSCAPE_FLAGS) -D -o $@ $<
endif

figs/%.png: figs/svg/%.svg
ifeq ($(INKSCAPE_VERSION_1),) # Inkscape < 1.0
    $(INKSCAPE) $(INKSCAPE_FLAGS) -D -d 300 -e $@ $<
else
    $(INKSCAPE) $(INKSCAPE_FLAGS) -D -d 300 -o $@ $<
endif
